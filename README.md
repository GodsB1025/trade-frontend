# 🤖 TrAI-Bot Frontend

**Vite**, **React**, **TypeScript**를 기반으로, **Tanstack Router**로 라우팅을 처리하고 **Tanstack Query**와 **Zustand**를 통해 상태를 관리하는 **TrAI-Bot** 프로젝트의 프론트엔드 레포지토리입니다.

사용자 인터페이스(UI)와 사용자 경험(UX)과 관련된 모든 코드를 포함하고 있습니다.
